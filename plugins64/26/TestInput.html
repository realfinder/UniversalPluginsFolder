TestInput


Author V. C. Mohan
Date 28 May 2020
EMail: mohanvc

TestInput plugin for Avisynth+ version, is free to use under GPL 3 licence.

TestInput plugin has two functions Pattern and Jitter. Pattern creates some pattern which may be useful to test software during development. Jitter introduces some Jitter on input, akin to what some VHS conversions have.

This plugin works in all avisynth+ version supported color spaces. 32 and 64 bit version plugins are provided.

Min, Max, Clamp

v0.1.1 (2004-03-19)

Variable-argument Min and Max functions for AviSynth.

by James D. Lin (stickboy)

Usage

Min(x1 [, x2 [, ...]])

Returns the minimum of its arguments. Min(4, 2.1, 9, 42) will return 2.1.

Max(x1 [, x2 [, ...]])

Returns the maximum of its arguments. Max(4, 2.1, 9, 42) will return 42.

Clamp(val, low, high)

Returns the value of val when constrained to the range [low, high]. Examples:

  • Clamp(4, 1.0, 9) will return 4
  • Clamp(13, 1.0, 9) will return 9
  • Clamp(0, 1.0, 9) will return 1.0

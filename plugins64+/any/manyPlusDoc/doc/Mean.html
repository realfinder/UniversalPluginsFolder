Mean
Author V. C. Mohan
Date 21 Oct 2020

Due to various causes noise is most often present on raw images. Averaging over a surrounding grid can attenuate noise at the cost of blurring. Some control to preserve edges is provided. All formats and bit depths are accepted. Option for not processing of U and V planes is included. This can also be used as a preprocessed example for SegAmp function.


This is thread safe. MT_NICE_FILTER
Details of parameters
Description Name Type Limits Default
Input clip to be smoothed clip none
size of square grid grid integer odd number between 3 to 15 5
tolerance tol float more than 0 to 1.0 0.1
u and v plane smoothing uv boolean true for yes, false for no true

Usage examples:-
Mean()
Mean(grid = 9,limit = 0.1, uv = false)

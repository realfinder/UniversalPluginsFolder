Lens

Lens function magnifies a specified circular disc area. The magnification can be constant in the disc or vary from center to edges ( drop=true). The lens center moves from the initial center coordinates in pixels to the final coordinates. RGB, YUY2 and Planar 8/ 9-16/32 bit depth formats are supported. Thread Safe


Details of parameters
Description Name Type Limits Default
clip clip none
start frame number sf Integer within clip first frame
end frame number ef Integer within clip last frame
Radius of lens radius integer 8 or more, less than half of smaller dimension of frame 1/8th smaller frame dimension
magnification at start magx integer 2 to 12 4
initial x coordinate x integer in frame mid frame
initial y coordinate y integer in frame mid frame
Final x coordinate ex integer in frame x
Final y coordinate ey integer in frame y
is drop effect required? drop boolean true for yes, false for no false

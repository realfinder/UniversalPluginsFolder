TVLook

The TVLook function on a base image creates a old TV set type image from a second clip (TV dimensions can be 16 : 9 or 4 : 3) . The color of cabinet and color bias as well as size , placement of the TV can be specified. The size and placement can be linearly varied along clip. As old CRT TV sets are brighter, may have some color bias, using col parameter this can be introduced. As they were non flat panels, curvature is introduced in the image. Quality of interpolation if set to high will run slower. . RGB, YUY2 and Planar 8/9-16/32 bit depths formats are supported. Thread Safe.


Details of parameters

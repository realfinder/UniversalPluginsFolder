Distort

Distort function distorts the image. RGB, YUY2 and Planar 8/ 9-16/32 bit depths formats are supported.Thread safe. Distortion is acheived by magnifying overlapping discs with water drop effect. "c[sf]i[ef]i[radius]i[mag]i[drop]b"


Details of parameters
Description Name Type Limits Default
clip clip none
start frame number sf Integer within clip first frame
end frame number ef Integer within clip last frame
radius radius integer 2 to 1/4th of frame smaller dimension 30
magnification mag integer 2 to 32 4
drop effect drop boolean true , false true

#Usage examples:- 
Distort()

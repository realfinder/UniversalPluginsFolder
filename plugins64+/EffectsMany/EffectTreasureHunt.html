EffectTreasureHunt

The image is scanned or totally processed using the code and sieve parameters to bringout hidden features. This is just for fun. Not all color spaces give identical results. RGB may be better.


Details of parameters
Description Name Type Limits Default
clip clip none
Effect start frame number sf Integer within clip first frame
Effect end frame number ef Integer within clip last frame
Is only scan desired? scan boolean true for scan only, false for full process true
your secret code code integer none internally set
sieve %age sieve integer 1 to 100 50

#Usage examples:- 
EffectTreasureHunt(last,50, 150, true, 400,40)
EffectTreasureHunt(last,50, 150, sieve = 40)
EffectTreasureHunt(last,50, 150,false)
EffectTreasureHunt(last,50, 150,code=375)

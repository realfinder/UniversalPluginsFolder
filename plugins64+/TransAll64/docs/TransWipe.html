TransWipe

Right Frame wipes away Left Frame in any of the eight directions. All avisynth supported formats can be input. RGB32, RGB24 YV12 and YUY2 colorspaces are supported by 2.5+ version.2.6 version supports all other planar formats as well.


Details of parameters
Description Name Type Limits Default
Left clip clip none
Right clip clip none
Overlapping frames or seconds time overlap integer not more than clip length , >2 frames. If -ve taken as time in seconds none
direction of wipe dir integer 1 to 8. 1 for north, 8 for nw in clockwise direction 3

#Usage examples:-
TransWipe(Left, Right,100,6)
TransWipe(Left, Right,-3)

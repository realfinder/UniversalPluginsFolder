TransShuffle

Left (first) clip moves out on top, while Right Clip moves from below in opposite direction halfway, then Right clip moves back to the top position while left clip slides under in opposite direction resembling movement of shuffling of cards. All avisynth supported formats can be input. RGB32, RGB24 YV12 and YUY2 colorspaces are supported by 2.5+ version.2.6 version supports all other planar formats as well.


Details of parameters
Description Name Type Limits Default
Left clip clip none
Right clip clip none
Overlapping frames or seconds time overlap integer not more than clip length , >2 frames. If -ve taken as time in seconds none
direction of shuffling dir string "up", "down", "right", "left" "up"

#Usage examples:-
Transshuffle(Left, Right,100,"left")
Transshuffle(Left, Right,-3)

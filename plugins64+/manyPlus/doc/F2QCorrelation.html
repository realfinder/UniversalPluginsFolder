F2QCorrelation


Author V. C. Mohan
1 Oct 2020

F2QCorrelation function cross correlates frames of 2 clips A and B, in frequency domain and requires FFTw3 32 bit dll or for 64 bit FFTw3 64 bit dll, to be in the path.

F2QCorrelation function has 2 input clips and dsplays either centered or raw cross correlation in image domain. Gamma enables display in proper perspective. The correlation maximum point coordinates frame wise can be output in a txt file. Note at end, the video file must be closed to get the file output.

Only Green color of RGB formats is used, in case of other formats Y plane is used for correlation.

It may be noted internally the input frame size may be increased for optimization of fft. The optimized size frame is output.

If txt is opted, a file is created for first 1000 frames only. The output gives Frame No, detected shift along x in search window, shift along y in search window, correlation maximum in the frame x coordinate and y coordinate. Normally both sets can be same, but in some cases different.


F2QCorrelation is thread safe.

Details of parameters
Description Name Type Limits Default
Input referance A clip clip must have the fields separated none
Input B clip clip must have identical format and dimensions of A clip none
gamma scaling for viewing convenience gamma float 4.0
Whether correlation text file to be generated. txt boolean true false. false
Full path and name of text file to be generated. file string a valid file name must be specified if txt is set true none
search along x for maximum xmax int less than 1/4 of frame width 20
search along Y for maximum ymax int less than 1/4 of frame width 20

#Usage examples:- 
# for text file output
F2QCorrelation(b,a, 4.0, true, txt = true, file = "C:\Documents\corr.txt")
#for visual output only
F2QCorrelation(b,a, 1.6, true)

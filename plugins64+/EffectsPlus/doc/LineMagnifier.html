LineMagnifier

LineMagnifier Magnifies either a vertical or Horizontal strip of image. The magnifier can be moved across image as per the coordinates. Mag is the magnification. RGB, YuY2 and Planar 8/9-16/32 bit depths only are supported. Thread safe.


Details of parameters
Description Name Type Limits Default
clip clip none
start frame number sf Integer within clip first frame
end frame number ef Integer within clip last frame
type of magnifier type string "hor" or "ver" "hor"
magnifier lens width lwidth integer 4 and 1/2 of frame width or height 1/8th of frame dimension
magnification mag integer 2 to 12 6
initial x or y coordinate of strip center xy integer lwidth/2 to width or height minus lwidth/2 lwidth/2
end xy coordinate exy integer lwidth/2 to width or height minus lwidth/2 width-lwidth/2

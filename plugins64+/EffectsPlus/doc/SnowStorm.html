SnowStorm

SnowStorm creates swirling white particles on the image. The snow can be either light, medium or heavy or increase from light to heavy or decrease from heavy to light. RGB, YUY2 and Planar 8/9-16/32 bit depth formats are supported.


Details of parameters
Description Name Type Limits Default
clip clip none
start frame number sf Integer within clip first frame
end frame number ef Integer within clip last frame
type of snow storm type string "heavy" "med" "light" "inc" "dec" "lnc"

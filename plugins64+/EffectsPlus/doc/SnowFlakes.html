SnowFlakes

SnowFlakes creates big or small size snow flakes that drift in wind and gently fall down. The dense (ness) of snow, drift %age, the falling speed ( %age) and size (big or small) of flakes can be opted. It is to be noted that the drift parameter is also affected by dense parameter and therefore not independent fully.(lesser dense greater drift for same %age). RGB, YUY2 and Planar 8/ 9-16/ 32 bit depth formats are supported. Thread Safe.


Details of parameters
Description Name Type Limits Default
clip clip none
start frame number sf Integer within clip first frame
end frame number ef Integer within clip last frame
density of snow fall %age dense integer 1 to 100 50
drift %age drift integer 1 to 100 10
Falling speed %age fall integer 1 to 100 20
are flakes big in size? big boolean true for big, false for small true
